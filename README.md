# SuperStore Sales Analysis using SQL

## How to Run the Project

1. Import the dataset (CSV file)
2. Run the SQL script
3. Explore the analysis queries

## Project Overview 
 This project analyzes retail sales data to uncover key insights related to sales 
 performance, profitability, customer segments, and discount strategies.

## Tools Used :
 -SQL
 -Excel 

## Data Cleaning Steps :
 -Removed duplicates using ROW_NUMBER
 -Standardized text fields
 -Converted data types
 -Handled missing values

## Key Insights & Recommendations

### KPI Performance
The dataset shows strong overall sales performance with a consistent average order value,
indicating stable customer purchasing behavior. This suggests maintaining current pricing
strategies while exploring opportunities to increase order volume.

### Top Products
Certain sub-categories dominate total sales, suggesting that focusing marketing efforts on
these high-performing products can significantly boost revenue. This suggests prioritizing 
inventory and promotions for these products.

### Low Performing Products
Some sub-categories generate very low sales, indicating potential issues in demand, pricing,
or product positioning. This suggests reviewing pricing strategies or considering
discontinuation of underperforming products.

### Regional Performance
Sales are heavily concentrated in specific regions, highlighting opportunities for expansion 
in underperforming areas. This suggests increasing marketing efforts in low-performing regions 
to drive growth.

### City Performance
A small number of cities contribute a large portion of total sales and profit, suggesting 
strong market potential in these locations. This suggests focusing targeted campaigns and 
customer retention strategies in these cities.

### Profitability Analysis
Some categories generate high sales but relatively lower profit, indicating inefficiencies
in pricing or higher operational costs. This suggests optimizing pricing strategies or
reducing costs to improve profitability.

### Discount Strategy
Higher discount levels are associated with lower average profit, indicating that excessive 
discounting negatively impacts profitability. This suggests limiting high discount levels and 
adopting more controlled discount strategies.

### Customer Segments
The Consumer segment generates the highest sales and order volume, making it the most valuable 
customer group for the business. This suggests focusing marketing efforts and loyalty programs 
on this segment.

Conclusion :
 This analysis highlights key opportunities to improve profitability by optimizing discount 
 strategies and focusing on high-performing products and customer segments.
